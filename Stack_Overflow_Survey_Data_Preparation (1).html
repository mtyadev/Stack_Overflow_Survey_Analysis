Preparing the Data¶

  • The relevant columns in the data set each contain multiple elements in a single cell separated by a semicolon.
  • e.g. programming languages = Ruby; Java; Python
  • Also sometimes there is a trailing white space contained before or after the semi colon.
  • For further analysis I will generate a function to split the data and remove trailing white spaces
In [15]:
def generate_columns_set(have,want):
    """Takes in columns belonging to relevant questions block (43-52). Splits and cleans the data.
    Generates set of all unique elements. This set can be used to setup a unique column by element (e.g. have used 
    language = python) in analyzing_have_want method below
    
    Args: 
        have (pandas series): Contains all elements currently in use by the respondent for a topic (e.g. all languages used)
        want (pandas series): Contains all elements the respondent wants to use in 2020 for a topic (e.g. all languages used)
    
    Returns:
        Set of all elements the respondent either is currently using or wants to use in the future
    """

    overview_set = set()
    for element_have, element_want in zip(have,want):
        # Adding all elements to overview set
        overview_set = overview_set | set(element_have.split(";")) | set(element_want.split(";"))
        # Cleaning overview set, since it contains white spaces before or after some of the strings
        overview_set = set([x.strip() for x in list(overview_set)])
    return overview_set

Modelling the Data¶

  • Now that I have all unique elements for all categories, I can use this information to set up an item interaction matrix by category.
  • The matrix will contain a column for each unique element, e.g. each unique language used by respondents in the data set.
  • The respondents will make up the rows of the matrix
  • In case a respondent has / wants to use one of the elements the matrix entry will be set to 1 for the respective column.
  • If the respondent hasn't/doesn't want to interacted with the element, the matrix will contain a 0 for this element.
In [17]:
def analyze_have_want(questionnaire_column,columns_set):
    """Takes in columns belonging to relevant questions block (43-52). Splits and cleans the data. The aim is to generate
    an item interaction matrix, showing 1 if a user has/wants to interact with an item and 0 if not.
 
    Args:
        questionnaire_column (pandas series) = Column with user responses for relevant question, e.g. "LanguageWorkedWith"
        columns_set (set) = Set of all unique elements in a category, e.g. all languages
    
    Returns:
            Item interaction Dataframe, showing 1 if a user has/wants to interact with a certain 
            tool/framework/language etc. and 0 if not.
    
    """

    # Setting up empty dataframe containing all unique elements by category as columns
    item_interaction = pd.DataFrame(columns = columns_set)
    
    # Adding column containing user response for relevant question e.g. "LanguageWorkedWith" to dataframe

    item_interaction[questionnaire_column] = have_want_work[questionnaire_column]

    # First filling all columns with 0's

    item_interaction.fillna(0, inplace = True)
    
    # Iterating over all element combinations and getting corresponding row_index of user in dataframe.
    for row_index, questionnaire_response in item_interaction[questionnaire_column].items():
        responses = questionnaire_response.split(";") if len(questionnaire_response.split(";")) > 0 else list() 
        # Cleaning elements list, since somteimes whitespaces are used before or after words
        responses = [x.strip() for x in responses]

        # Checking unique elements in questionnaire response and setting related column in item_interaction df to 1
        for element in responses:

            # Iterating over all columns in column_set. Checking if given current element matches column. 
            for column_name in columns_set:
                if element == column_name:
                    item_interaction.at[row_index, column_name] = 1  
                    
    return item_interaction

def find_matches(have,want,finding_matches=finding_matches):
    """Conducts a pairwise comparison of all elements in provided two pandas series.
    
    Args:
        have (pandas series): Contains all elements currently in use by the respondent for a topic (e.g. all languages used)
        want (pandas series): Contains all elements the respondent wants to use in 2020 for a topic (e.g. all languages used)
        finding_matches (pandas dataframe): Contains a copy of the to be analyzed dataframe
    Returns:
        Sum of all exact (non-)matches of the pairwise comparison   
    """
    
    # Generating lookup column which will display whether or not the pairwise comparison is an exact match
    finding_matches["Matching_Result"] = np.where(
        finding_matches[have].apply(
        lambda x : set(element.strip() for element in x.split(";")) if len(x) >= 1 else "Not Matching") == 
        finding_matches[want].apply(
        lambda x : set(element.strip() for element in x.split(";")) if len(x) >= 1 else "Not Matching")  
        , "Matching","Not Matching")
    
    return finding_matches["Matching_Result"].value_counts()

def generate_relative_comparison_df(have,want):
    """
    Generates a dataframe showing a pairwise comparison of current and desired situation by category.
    
    Args: 
        have (pandas series): Contains all elements currently in use by the respondent for a topic (e.g. all languages used)
        want (pandas series): Contains all elements the respondent wants to use in 2020 for a topic (e.g. all languages used)  
    
    Returns: 
        Dataframe containing a pairwise comparison of current and desired situation by category.
    """
       
    # Initializing dataframe based on provided input 
    comparison_df = pd.concat([have,want], axis=1,sort=False)
    comparison_df.columns = ["Have","Want"]
    # Calculating relative comparison of have and want in percent
    comparison_df["Have"] = (comparison_df["Have"] / comparison_df["Have"].sum(axis = 0)) * 100
    comparison_df["Want"] = (comparison_df["Want"] / comparison_df["Want"].sum(axis = 0)) * 100
    return comparison_df
